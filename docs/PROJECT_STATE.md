## 3. 계획 중인 메뉴 구조

현재 메뉴는 단순 4항목(추천게임찾기 / 전체게임보기 / 매장이용안내 / 문의하기).  
아래 구조로 확장 예정:

```
추천 게임 찾기
전체 게임 보기
코티지보드
  ㄴ 소개 (정체성 + 공간 특성)
  ㄴ 동호회
이용 안내
  ㄴ 가격 & 규칙
  ㄴ 게임 위치
  ㄴ 오시는 길
  ㄴ 문의
```

**필요한 신규 페이지 (미작성)**

| 경로(예정) | 내용 |
|-----------|------|
| `about.html` 또는 `index.html#about` | 코티지보드 소개, 공간 특성 |
| `club.html` | 동호회 안내 |
| `store-usage-guide.html` | 가격 & 규칙 ← 현재 플레이스홀더 |
| `game-location.html` | 게임 위치/서가 안내 |
| `map.html` 또는 섹션 | 오시는 길 (지도) |
| `contact.html` | 문의 ← 현재 플레이스홀더 |

**메뉴 구현 방식 미결정 사항**

- 드롭다운 서브메뉴 구현 여부
- 현재 메뉴는 모바일 오버레이 방식(`.header-menu`) — 서브메뉴 추가 시 구조 변경 필요
- "코티지보드"와 "이용 안내" 섹션은 그룹 헤더인지 링크인지 결정 필요
